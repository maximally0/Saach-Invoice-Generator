You are building a highly user-friendly, production-ready invoice generator web app using the provided files (PDF, CSV/Excel reference, and branded HTML).

This tool will be used by non-technical users (sales teams, operators). It must feel extremely simple, clean, and impossible to mess up.

---

### 🎯 CORE GOAL

Transform the provided HTML invoice into a smart, interactive, and visually clean invoice generator that:

* Requires ZERO training to use
* Looks premium and modern
* Prevents user errors
* Works smoothly on desktop and mobile

---

### 🧠 UX PRINCIPLES (VERY IMPORTANT)

Follow these strictly:

1. **Big, clear inputs**

   * Large input fields
   * Clear labels (not placeholders only)
   * Good spacing
   * No cramped UI

2. **Minimal thinking required**

   * User should not calculate anything
   * Everything auto-updates instantly

3. **Guide the user step-by-step**
   Add section headings:

   * "Client Details"
   * "Invoice Details"
   * "Products"
   * "Summary"

4. **Error-proof**

   * Prevent empty or invalid inputs
   * Default values where possible
   * Show warnings (e.g. "Qty missing")

5. **Instant feedback**

   * Totals update live
   * Row amount updates instantly

6. **Make it feel fast**

   * No lag
   * No reloads

---

### 🧾 FORM STRUCTURE

Organize into clean sections:

#### 1. Client Details

* Client Name (required)

#### 2. Invoice Details

* Invoice Number (auto-suggest or editable)
* Date (default = today)
* Order Number (optional)

#### 3. Products (MOST IMPORTANT SECTION)

* Each row should have:

  * Product Name
  * Composition
  * Form
  * Packing
  * Qty
  * Rate
  * Amount (auto-calculated, read-only)

* Features:

  * "Add Product" button
  * "Remove Row" button
  * Minimum 1 row always present

---

### 💰 CALCULATION LOGIC (STRICT)

Follow EXACTLY:

* amount = qty × rate

* productSubtotal = sum of all amounts

* cylinderCharges = 1500

* inventoryCharges = 3000

* gstProducts = 5% of productSubtotal

* gstInventory = 18% of inventoryCharges ONLY

DO NOT:

* apply 18% GST to products

* apply GST to cylinder charges

* grandTotal = productSubtotal + cylinderCharges + inventoryCharges + gstProducts + gstInventory

* advance = 30% of grandTotal

All values must:

* auto-update instantly
* never show NaN
* always show 2 decimal places

---

Use:
