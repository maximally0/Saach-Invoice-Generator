You are building a production-ready web app using the provided files:

* SAACH Proforma Invoice PDF (reference for correctness)

Your goal is to convert the HTML invoice into a fully functional invoice generator web app.

---

### CORE FUNCTIONALITY

The app must allow a user to:

1. Fill in invoice details (client name, invoice number, date, order number)
2. Fill in product rows (product name, composition, form, packing, qty, rate, artwork)
3. Automatically calculate all financial values
4. Download:

   * A properly formatted PDF invoice
   * A CSV file of the invoice data

---

### CRITICAL: CALCULATION LOGIC (MUST BE EXACT)

Follow this EXACT logic from the original invoice:

1. Product Amount per row:
   amount = qty × rate

2. Product Subtotal:
   sum of all product amounts

3. Fixed Charges:
   cylinderCharges = 1500
   inventoryCharges = 3000

4. GST:
   gstProducts = 5% of productSubtotal
   gstInventory = 18% of inventoryCharges ONLY

   IMPORTANT:

   * Do NOT apply 18% GST to products
   * Do NOT apply GST to cylinder charges

5. Grand Total:
   grandTotal = productSubtotal + cylinderCharges + inventoryCharges + gstProducts + gstInventory

6. Advance:
   advance = 30% of grandTotal

All values must update in real-time when inputs change.
